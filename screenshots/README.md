# 📚 AI eBook Creator

An AI-powered MERN stack application that helps users generate, edit, manage, and export professional eBooks. Users can create books with AI-generated content, edit chapters using a markdown editor, upload custom cover images, and export books as PDF or DOCX.

## ✨ Features

- 🤖 AI-powered eBook generation
- 📖 Chapter-based content management
- ✍️ Markdown editor with live preview
- 🖼️ Custom cover image upload
- 📄 Export as PDF
- 📝 Export as DOCX
- 🔐 JWT Authentication
- 👤 User Profile Management
- 🔍 Search and filter books
- 📱 Responsive design
